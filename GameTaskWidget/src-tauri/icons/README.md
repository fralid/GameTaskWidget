# Иконки приложения

Для сборки приложения нужны иконки в следующих размерах:

- `32x32.png`
- `128x128.png`
- `128x128@2x.png` (256x256)
- `icon.icns` (macOS)
- `icon.ico` (Windows)

Можно использовать онлайн-генераторы или создать иконки вручную. Для разработки можно использовать временные заглушки.
